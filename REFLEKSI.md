# Refleksi - Tugas Pra-PBL 2026

## Pembelajaran yang Didapat

### Git & GitHub
- Memahami cara kerja Git branching strategy (main, develop, feature/*)
- Menerapkan Conventional Commit untuk pesan commit yang terstruktur
- Memahami alur Pull Request dan code review
- Menerapkan Semantic Versioning untuk manajemen rilis

### Backend (ASP.NET)
- Memahami cara membuat REST API menggunakan ASP.NET Core
- Menerapkan Entity Framework Core untuk ORM dan migrasi database
- Memahami konsep soft delete untuk menjaga data historis
- Menerapkan validasi input menggunakan DataAnnotations

### Frontend (React + TypeScript)
- Memahami cara membuat tampilan CRUD menggunakan React
- Menerapkan TypeScript untuk type safety
- Memahami cara komunikasi frontend dengan backend menggunakan Axios

### Mobile (Flutter)
- Memahami cara membuat aplikasi mobile menggunakan Flutter
- Menerapkan HTTP request menggunakan package http
- Memahami cara menampilkan data dari API di aplikasi Flutter

## Tantangan yang Dihadapi
- Konfigurasi database MySQL dengan ASP.NET Core
- Mengatasi CORS error antara frontend dan backend
- Kompatibilitas versi Node.js dengan Vite
- Encoding karakter UTF-8 pada file TypeScript

## Kesimpulan
Tugas Pra-PBL ini memberikan pengalaman nyata dalam pengembangan
perangkat lunak profesional menggunakan standar industri. Mulai dari
pengelolaan kode dengan Git, implementasi CRUD dengan ASP.NET,
hingga pembuatan tampilan dengan React dan Flutter.
