# Studi Kasus - Sistem Peminjaman Ruangan Kampus

## Latar Belakang
Ruangan kampus digunakan untuk berbagai kegiatan akademik dan non-akademik.
Saat ini pencatatan peminjaman masih dilakukan secara manual melalui chat,
catatan, atau spreadsheet, sehingga pengelolaan data menjadi tidak terpusat.

## Permasalahan
- Data peminjaman mudah tercecer
- Status peminjaman sulit dipantau
- Riwayat penggunaan ruangan tidak terdokumentasi dengan baik
- Tidak tersedia data rapi untuk evaluasi

## Solusi
Membangun sistem peminjaman ruangan kampus berbasis web dan mobile yang
memungkinkan pengelolaan data peminjaman secara digital dan terpusat.

## Target Pengguna
- Admin kampus yang mengelola peminjaman ruangan
- Mahasiswa/dosen yang mengajukan peminjaman ruangan

## Manfaat
- Data peminjaman tersimpan terpusat dan aman
- Status peminjaman dapat dipantau secara real-time
- Riwayat penggunaan ruangan terdokumentasi dengan baik
- Tersedia data yang rapi untuk evaluasi dan pelaporan
